# Hybrid Encryption Suite

A collection of cryptography projects demonstrating secure hybrid encryption using AES-256-GCM and RSA-2048.

## Projects Included

### 1. Text Encryption Tool

A Flask-based application that encrypts and decrypts text using hybrid cryptography.

**Features**

* AES-256-GCM encryption
* RSA-2048 key exchange
* Secure text protection
* Web-based interface

### 2. Image Encryption Tool

A Flask-based application that encrypts and decrypts image files using hybrid cryptography.

**Features**

* AES-256-GCM image encryption
* RSA-2048 key protection
* JSON export support
* Secure image recovery
